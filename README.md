# Voxel Engine

웹 기반 고성능 복셀 게임 엔진

## 목표

- **플랫폼**: 웹 중심 (데스크톱 웹 > 모바일 웹)
- **복셀 규모**: 1024 × 1024 × 512 (~5.4억 복셀)
- **동적 수정**: 실시간 블록 추가/삭제
- **성능**: 마인크래프트 이상의 FPS

## 기술 스택

| 레이어 | 기술 |
|--------|------|
| 언어 | TypeScript |
| 렌더링 (메인) | WebGPU |
| 렌더링 (폴백) | WebGL 2.0 |
| 병렬 처리 | Web Workers |
| 빌드 | Vite |
